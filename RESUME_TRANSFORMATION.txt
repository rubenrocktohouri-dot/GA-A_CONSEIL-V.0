💾 MODÈLES DE DONNÉES CRÉÉS
═══════════════════════════════════════════════════════════════════════════════

  SensorData (Télémétrie)
  ├─ drone_id (String)
  ├─ farmer_username (FK)
  ├─ humidity (Float) %
  ├─ temperature (Float) °C
  ├─ soil_ph (Float)
  ├─ soil_moisture (Float) %
  ├─ nitrogen_level (Float) mg/kg
  └─ timestamp (DateTime)

  Alert (Alertes Critiques)
  ├─ alert_type (String) - epidemic, drought, disease, frost
  ├─ severity (String) - critical, high, medium, low
  ├─ farmer_username (FK)
  ├─ message (Text)
  ├─ data_snapshot (JSON)
  ├─ activated (Boolean)
  └─ created_at (DateTime)

  AdminRecommendation (SAD)
  ├─ admin_username (FK)
  ├─ farmer_username (FK)
  ├─ content (Text)
  ├─ based_on_sensor_data (Boolean)
  ├─ created_at (DateTime)
  └─ read (Boolean)

  DroneFleet (Gestion Flotte)
  ├─ drone_id (String, unique)
  ├─ farmer_username (FK)
  ├─ status (String) - online, offline, maintenance
  ├─ battery_level (Float) %
  ├─ location (String)
  └─ last_connection (DateTime)


🔐 SÉCURITÉ IMPLÉMENTÉE
═══════════════════════════════════════════════════════════════════════════════

  ✅ Authentification Forte
     • Décorateur @admin_required
     • Vérification du rôle utilisateur
     • Tokens SHA-256

  ✅ Validation des Données
     • Sanitisation HTML (escapeHtml)
     • Vérification des paramètres
     • Gestion des erreurs 400/403/404

  ✅ Isolation des Flux (Protocole RELYAS)
     • Routes admin ≠ Routes publiques
     • Planteurs ne voient que leurs données
     • Admin voit tout

  ✅ Sessions Sécurisées
     • HTTPOnly cookies
     • SameSite protection
     • Tokens d'authentification
